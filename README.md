# w51-e2w-model.

Run everything via:

```bash
python generate_master.py
python analysis/parabola.py
python analysis/parabola_model.py
```

## Outputs:

* sed_all.png
* global_gridA.png
* global_gridB.png
* global_gridC.png
* data/image_45ghz.fits
* data/image_95ghz.fits
* data/image_45ghz-conv.fits
* data/image_95ghz-conv.fits
* radmc3d_inp/
* radmc3d_out/
